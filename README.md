Library REST API

This is a RESTful API for a simple library, developed using Node.js, Nest.js, and MongoDB.

Tech Stack:

Backend: Node.js, Nest.js, MongoDB

Authentication: JWT (JSON Web Tokens)

Authorization: Role-based access control (RBAC)
